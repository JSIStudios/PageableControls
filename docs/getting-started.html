2. Requirements / dependencies

jQuery

These controls were built and tested with jquery 1.9.1 and may not work with earlier versions (< v1.9) due to breaking changed introduced in the v1.9.x described here.

Project home Download v1.9.1

Twitter bootstrap [optional]

The default behavior for these controls utilize the styles and functionality from Bootstrap v2 (support for v3 will be available shortly). However, this library is not required and the dependent styles can be overwritten.

Project home Download v2.3.2

Make sure you add the dependencies on the page before the project files.

<link rel="stylesheet/less" href="/css/bootstrap.min.css">
<link rel="stylesheet/less" href="/css/pageablecontrols.min.css">
<script src="/js/jquery.min.js"></script>
<script src="/js/bootstrap.min.js"></script
<script src="/js/pageablecontrols.min.js"></script>

3. Project contents
